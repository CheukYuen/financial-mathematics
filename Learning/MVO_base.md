## 🧭 一、什么是 MVO（Mean–Variance Optimization）

> **均值–方差优化（MVO）** 是现代投资组合理论（Markowitz 1952）的核心方法，
> 目标是在 **“期望收益” 与 “风险”** 之间找到最优平衡。

简单一句话：

> “在预期收益给定的情况下，把风险（波动）降到最低；
> 或在风险容忍范围内，让收益最高。”

---

## ⚙️ 二、MVO 的输入与输出

| 类别               | 变量         | 含义                       | 示例                          |
| ---------------- | ---------- | ------------------------ | --------------------------- |
| **输入 (Inputs)**  | μ（mu）      | 各资产的**期望收益率向量**          | 现金 2%、债券 4%、股票 8%、黄金 5%     |
|                  | Σ（Sigma）   | 各资产之间的**协方差矩阵**（风险与相关性）  | 资产间同涨同跌的程度                  |
|                  | 约束条件       | 权重 ∑w=1、不能卖空 w≥0、或单资产上限等 | 股票 ≤ 50%，现金 ≥ 5%            |
| **输出 (Outputs)** | w*（最优权重向量） | 每个资产的配置比例                | 现金 10%、债券 40%、股票 40%、黄金 10% |
|                  | 风险–收益坐标    | 有效前沿（Efficient Frontier） | 各最优组合的风险–收益曲线               |
|                  | 最优投资组合     | 在给定风险/收益目标下的最优点          | “最小方差组合”或“最大夏普比组合”          |

---

## 🎯 三、MVO可以得到什么？

| 成果                                         | 说明                                 |
| ------------------------------------------ | ---------------------------------- |
| **1️⃣ 最优权重（w*)**                           | 给出每种资产应投资多少比例，形成风险–收益平衡组合。         |
| **2️⃣ 有效前沿（Efficient Frontier）**           | 一条“同风险最高收益 / 同收益最低风险”的曲线，显示所有最优组合。 |
| **3️⃣ 最小方差组合（Minimum Variance Portfolio）** | 在所有组合中风险最低的那个点。                    |
| **4️⃣ 最大夏普比组合（Tangency Portfolio）**        | 风险调整后收益最高的点（与无风险资产连线的切点）。          |
| **5️⃣ 投资者个性化配置**                           | 根据风险偏好选取有效前沿上的不同点。                 |

---

## ⚠️ 四、MVO 的关键假设与注意事项

| 类别        | 内容                          | 风险或问题                          |
| --------- | --------------------------- | ------------------------------ |
| **模型假设**  | 收益服从正态分布；风险用方差衡量            | 实际收益往往偏态、厚尾，方差未必完全代表风险         |
| **输入稳定性** | μ 和 Σ 对结果极敏感                | 小变化会导致权重大幅波动（“不稳健性”）           |
| **现实约束**  | 存在交易成本、最小申购量等               | 理论最优组合可能无法完全执行                 |
| **优化陷阱**  | 过度拟合历史数据                    | 未来 ≠ 过去；需要稳健化（Shrinkage, BL修正） |
| **改进方向**  | Black–Litterman、风险平价、贝叶斯更新等 | 在实际投顾系统中常用这些方式改进MVO的输入         |

---

## 🧩 五、可以直接放进 PPT 的一句话总结

> **MVO 是投资组合优化的数学核心：**
> 它把“期望收益（μ）”与“风险（Σ）”作为输入，
> 通过优化算法得到最优权重（w*），
> 绘制出有效前沿，帮助投资者在风险与收益间做权衡决策。

## 🔍 七、专业补充（给懂行听）

> 数学目标函数：
> [
> \text{minimize } w^T Σ w \quad s.t. \quad w^T μ = μ^*, \sum w_i = 1, w_i ≥ 0
> ]
> 输出 w*。在数值实现中常用 `cvxpy`、`PyPortfolioOpt`、`scipy.optimize`。
